

## 实现线程切换

### 环境

如果`make`之后出现无法编译的情况, 那么可能缺少了`-m32`条件的环境.

#### 解决

缺少链接32位的环境, ubuntu中需要下载

```shell
sudo apt-get install build-essential module-assistant
sudo apt-get install gcc-multilib g++-multilib
```

需要使用32位环境时, 在gcc选项中加上`-m32`就行.
